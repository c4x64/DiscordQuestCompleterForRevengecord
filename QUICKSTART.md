# Quest Auto-Completer - Quick Start

## 🚀 Installation (30 seconds)

1. **Open Revenge Cord** on your Discord mobile app
2. **Go to**: Settings → Plugins
3. **Paste this URL**: (You'll host your own after deployment)
   ```
   https://YOUR_USERNAME.github.io/QuestAutoCompleterPlugin/quest-auto-completer
   ```
4. **Click Install** and wait for it to load
5. **Restart Discord** to apply changes
6. **Go to Revenge Settings** and find "Quest Auto-Completer"

## ⚡ Main Features

### Quick Actions
- **⚡ Complete All** - Finish all available quests instantly
- **🔄 Refresh** - Update the quest list
- **⚙️ Settings** - Configure your preferences

### Per-Quest Control
- **Tap any quest** to see details
- **Click the button** to complete individual quests
- **View progress bars** for each quest

### Smart Settings
- **Auto Refresh** - Automatically reload quests every 10 seconds
- **Show Completed** - Display already finished quests
- **Confirm Before** - Get a confirmation dialog before completing
- **Notifications** - Toast notifications on quest completion

## 📊 Dashboard

The plugin shows real-time stats:
- **Available**: Quests waiting to be completed
- **Completed**: Finished quests
- **Total**: All quests combined

## 🔧 Troubleshooting

### "Quest module not found" error
- Make sure you're on the latest Revenge version
- Restart Discord completely
- Try reinstalling the plugin

### Quests not showing up
- Tap the **Refresh** button
- Make sure you have active quests
- Check your internet connection

### Plugin not installing
- Verify the URL is correct
- Check your internet connection
- Try restarting Discord

## ✨ Pro Tips

1. **Disable confirmations** in settings for speed
2. **Enable auto-refresh** to always see new quests
3. **Complete all at once** when you have multiple quests
4. **Check completed quests** toggle to see what you've done

## 📱 System Requirements

- **Discord Mobile** (Android or iOS)
- **Revenge Cord** v250.10+ (Sept 28, 2024+)
- **Internet connection**
- **Some available quests**

## 🎯 What It Does

This plugin:
- ✅ Automatically fills quest progress to 100%
- ✅ Submits quest completion to Discord
- ✅ Shows real-time quest information
- ✅ Provides individual or bulk completion options
- ✅ Persists settings between restarts

## ❌ What It DOESN'T Do

- ❌ Complete quests that are already done
- ❌ Modify your account or messages
- ❌ Work offline
- ❌ Work with custom quests
